# Calendar App In React Js

Calendar App In React Js React Big Calendar  and json server for storing and show events

## Resources used
- react js
- json server
- axios
- react-big-calendar
- react-date picker
- concurrently
## Installation

you may use **Yarn** or **Npm** to initialize and install all required dependencies

### for yarn run 

`yarn install` 

### for npm run 

`npm install`
